# 俄罗斯方块小游戏

## 控制台版本

> c语言版本 2022.4.28

使用 20 * 10 的地图

7 种方块， 19 种变换模式.

### 使用方法

* w 变换

* a 左移

* d 右移

* 空格 方块直接落地


游戏没有胜利条件 无尽模式
当游戏中方块落地后，y坐标<0 时，我们可以认定游戏失败

## easyX 修订版本

> 为了学习easyX ,迫不得已更新了控制台版本的 俄罗斯方块  2022.4.29

> 使用 20 * 10 的地图 

矩形框为方块的载体，原本想模仿远古游戏机的界面，没画出来对应的效果。。凑合吧。

做了UI 和逻辑分离，简单改改UI 就能上线。

### 使用方法

* w 变换

* a 左移

* d 右移

* 空格 方块直接落地

* p 暂停游戏

* r 重玩游戏  需要按 y 进行确认



游戏没有胜利条件 无尽模式
游戏失败时，会确认是否重新开始游戏。y重新开始。
